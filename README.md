# KnowYourPew

Church member connection platform. Built for Woodridge Baptist Men's Ministry, designed to scale to any church.

## Vision
A purpose-built web app that helps churches actually know their people. Progressive profiling, interest-based matchmaking, and a modern member directory.

## Tech Stack
- **Frontend:** Next.js 14 + Tailwind CSS + shadcn/ui
- **Database:** Supabase (Postgres + Auth + Storage)
- **Hosting:** Vercel (free tier)
- **Domain:** knowyourpew.com

## Phases
- **Phase 1:** Signup form + admin dashboard
- **Phase 2:** Member login + directory + profile browsing
- **Phase 3:** Spiritual gifts assessment + deeper profiling
- **Phase 4:** Interest groups, events, messaging
